Instructions
How To Explore
Click on the FPGA square to see per-tick TX and RX packet activity at the FPGA.
The popup shows the bit on the current tick, the packet word, and decoded packet fields, which is especially useful for configuration packets.
How To Run
Use Play and Pause and Step Back and Step to see a tick-accurate recreation of network simulator results.
You can also upload playback JSON files produced by other simulation runs.
Packet Loss
Scroll down in the HUD to the Data Packet Metrics section to see packet generation, FPGA arrival counts, and packet-loss-related summary information.
Run Metrics
Click the Run Metrics button to open the popup with ticks per second and total runtime for the run.
