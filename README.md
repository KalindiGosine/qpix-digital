# qpix-digital GitHub Pages export

This directory contains a self-contained copy of the LArPix visualizer and selected playback JSON files for deployment to:

- Repo: `https://github.com/KalindiGosine/qpix-digital`
- Pages URL: `https://kalindigosine.github.io/qpix-digital/`

## What to push

Copy the contents of this directory to the root of the `qpix-digital` repository and push to `main`.

## Enable GitHub Pages

On GitHub: `Settings -> Pages -> Build and deployment -> Deploy from a branch -> main / (root)`.

## Default scenario

The default playback is `./playback/live_event_3x5_chip14.json`.
